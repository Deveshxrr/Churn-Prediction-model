# Customer Churn Prediction & Behavioral Analytics

This project focuses on understanding customer behavior and identifying customers who are likely to stop using a service or product.

The project uses customer activity data such as login history, payment delays, complaints, discounts, purchases, and usage patterns to perform customer analysis and churn prediction.

---

## Project Objectives

- Analyze customer activity and engagement
- Identify high-value customers
- Detect delayed payment users
- Find customers with frequent complaints
- Perform customer segmentation
- Predict customer churn using Machine Learning

---

## Tools & Technologies Used

- Python
- SQL (SQLite)
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Project Workflow

1. Load multiple customer datasets
2. Store and manage data using SQLite
3. Clean and preprocess data
4. Perform customer behavior analysis
5. Create customer segments using K-Means Clustering
6. Detect payment risk and complaint patterns
7. Build churn prediction model using Random Forest
8. Visualize important business insights

---

## Machine Learning Model

- Random Forest Classifier

### Model Accuracy
- 88.95%
